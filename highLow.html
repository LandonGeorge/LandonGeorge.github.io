High/Low Game

Technologies/Software used: PHP, Bootstrap, MySQL, HTML5, CSS3, and PhpStorm

In this individual project I utilized PHP for server-side code in order to hide some parts of the game logic. One specific part that is hidden from the user on the front-end is the number they are trying to guess, which is randomly generated server-side. This application also does user account creation and authentication utilizing a MySQL database and SHA256 hashing in order to protect user passwords. One particular challenge I faced during this project was displaying a high score table of the top 10 scores when the user won. I overcame this challenge by utilizing PHP to fetch data from my database and display it in a table using a while loop to create new rows for each score entry. In addition this website is hosted via InfinityFree with SSL enabled and can be visted with the button below!

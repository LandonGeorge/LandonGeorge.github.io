Hangman Game

Technologies used: .NET, Angular, Enity Framework, AJAX

This project was a group project in which we created a asynchronous hangman game complete with account creation, account authentication and high score tracking. In this project I created the logic that pulls a random word from a file. This was challenging because I never grabbed words from a file before in a program. I was able to come up with a solution using my previous experience in C# and sample code from the internet.

Random Word From File Code


//Generate new word
public void GenerateWord()
{
	Random rand = new Random();
	List words = new List();
	using (var sr = new StreamReader("./Data/words.txt"))
	{
		var endOfFile = false;
		while (!endOfFile)
		{
			var word = sr.ReadLine();
			
			if (word != null && !words.Contains(word))
			{
				words.Add(sr.ReadLine());
			}
			else if(word == null)
			{
				endOfFile = true;
			}
		}
	}

	var randomIndex = rand.Next(0, words.Count()-1);

	SetWord(words[randomIndex]);
}
